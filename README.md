# CUPHY

CUPHY is a premium Physics EdTech platform built with React, Vite, Firebase, Firestore, Firebase Functions, and Firebase Hosting.

## Features

- Phone-first authentication
- Email-linked student login
- OTP signup and forgot password
- Admin panel
- Subscription unlock system
- Notifications
- Support tickets
- Tests and results
- Premium UI

## Project Structure

- `src/` → React frontend
- `functions/` → Firebase Functions
- `public/` → static assets
- `firebase.json` → Firebase config
- `firestore.rules` → Firestore security rules

## Run locally

```bash
npm install
npm run dev
